API project presentation

Хочу презентовать вам мой проект консультанта интернет-магазина для голосового помощника Алисы.
<next>
Это нечто иное как чат-бот, реализованный через API, который может советовать и искать товары для пользователей.
<next>
Также в нем возможна и авторизация, после которой становится возможным просмотр корзины, добавление туда товаров и создание заказов.
<next>
Теперь о реализации. Код построен таким образом, что есть функция, которая принимает http запрос с данными в формате JSON. Далее идёт обработка этих данных, и вызывается функция ведения диалога для формирования нужного ответа.
Этот самый ответ создается исходя из текущего состояния диалога с пользователем и, соответственно, переданного от него сообщения.
Насчёт состояния диалога – когда формируется ответ со стороны сервера, позволяющий пользователю логично на него тоже ответить, записывается соответствующее состояние, чтобы потом понять, к чему отнести следующее сообщение.
Отдельно вынесена функция для добавления в ответ кнопок-подсказок пользователю в зависимости от, опять же, текущего состояния.
